Check out our free REST API


After creating an account and logging in you can visit http://football-assistant.herokuapp.com/api/footballClubs and many more free endpoints to obtain all the data we provide in a slick RESTfull manner!

Currently supported endpoints:

  • Football Clubs: http://football-assistant.herokuapp.com/api/footballClubs
  • Leagues: http://football-assistant.herokuapp.com/api/leagues
  • Players: http://football-assistant.herokuapp.com/api/players
